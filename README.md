# 🧮 Calculator Windows Forms Project

## 🧾 Description
This is a simple Calculator application built using C# Windows Forms.  
It allows users to perform basic arithmetic operations like addition, subtraction, multiplication, and division.

---

## 🚀 Features
- Basic arithmetic operations (+, -, *, /)
- Simple and user-friendly interface
- Real-time expression input
- Display result instantly
- Error handling for invalid calculations

---

## 🛠️ Technologies Used
- C#
- .NET Framework
- Windows Forms
- DataTable.Compute (for evaluating expressions)

---

## ⚙️ How It Works
- Users click buttons to build a mathematical expression
- Expression is displayed in a TextBox
- On pressing "=" the result is calculated and shown

---

## 📸 Buttons Included
- Numbers (0–9)
- Operations (+, -, *, /)
- Clear (C)
- Equal (=)
